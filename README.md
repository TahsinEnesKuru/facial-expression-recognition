### Demo UI:

1) streamlit kütüphanesini kuralım:

```
pip install streamlit
```

2) Yukarıda paylaşılan modelleri indirip; models/ klasörünün altında toplayalım.

3) Demo uygulamayı run edelim:

```
streamlit run demo.py
```
